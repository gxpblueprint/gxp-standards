# Glossary of Terms

| Term | Definition |
| :--- | :--- |
| **CSA** | Computer Software Assurance. A risk-based approach to CSV focusing on critical thinking. |
| **ALCOA+** | Data Integrity principle: Attributable, Legible, Contemporaneous, Original, Accurate. |
| **GxP** | "Good Practice" regulations (GMP, GDP, GLP, GCP). |
